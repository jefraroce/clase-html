QUE ES CSS

CSS, Hojas de estilo en cascada o Cascading Style Sheets en ingles, es un lenguaje que permite espeficicar como se muestran los docuemntos HTML a los usuarios.

Permite organizar la presentación de un documento HTML, con CSS podemos definir el estilo de letra, alineación del contenido, los colores, etc.

Ejemplo de CSS

CSS se basa en separar lo que es la estructura de un documento HTML de su presentación, por ejemplo:

Tenemos un libro, el libro contiene la información que es el contenido del mismo, eso es HTML, ahora imaginemos que queremos que la caratula del libro sea de color rojo, que el texto sea más grande, las hojas del libro seán azules, eso es CSS.
