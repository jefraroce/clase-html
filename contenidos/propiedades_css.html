Propiedades CSS

CSS permite cambiar la apariencia de una página web.

Texto

Color texto

Permite cambiar el color del texto. 

            <p style="color: green;" > Este es un párrafo </p>

Font-size

Font-family

Permite cambiar el tipo de letra.
