Agrupación de Contenidos

P

Define un párrafo. 
<p>Este es un párrafo</p>

Este es un párrafo

Hr

Representa una separación de contenido. 
                            
<p>Este es un párrafo</p>
<hr/>
<p>Este es otro párrafo</p>

Este es un párrafo

Este es otro párrafo

Pre

Representa una texto preformateado y el formato es preservado. 
                            
<pre>
    Hola
        Este es un texto preformateado
    Adios
</pre>
                            
                        
    Hola
        Este es un texto con preformateado
    Adios

Ol

Representa una lista ordenada. 
                            
<ol>
    <li>Item uno.</li>
    <li>Item dos.</li>
    <li>Item tres.</li>
    <li>Item cuatro.</li>
</ol>
  1. Item uno.
  2. Item dos.
  3. Item tres.
  4. Item cuatro.

Ul

Representa una lista sin orden. 
                            
<ul>
    <li>Item uno.</li>
    <li>Item dos.</li>
    <li>Item tres.</li>
    <li>Item cuatro.</li>
</ul>
  • Item uno.
  • Item dos.
  • Item tres.
  • Item cuatro.

<div>
    <p>Este es un párrafo dentro de un div.</p>
</div>
